# ng-samskriti
Ng-Samskriti is an exploration of the modern web development process with the <a href="https://angular.io/">Angular framework</a>.

This project aims to deliver content about the Sanskrit language and Hindu philosophy.

# Current - State
The project is currently not hosted publicly yet. To take a look at the project, install the <a href="https://cli.angular.io/">Angular CLI</a> and use the `ng serve` to host the site locally.
